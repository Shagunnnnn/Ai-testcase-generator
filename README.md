# Ai-testcase-generator
AI-powered web application that automatically generates software test cases using LLMs.

## Tech Stack
- Python
- Flask
- MongoDB
- JavaScript
- Ollama/Groq API

## Demo Video
[Watch Demo](https://drive.google.com/file/d/1LnWAesEUlWn72bkA2aVV1HNH-XTdHa7-/view?usp=drive_link)

## Features

### 1. AI-Based Test Case Generation
Generates professional manual QA test cases automatically from user requirements using LLMs like Ollama/Groq.

### 2. Requirement Analysis
Analyzes functional requirements provided by the user and converts them into structured testing scenarios.

### 3. Automated QA Workflow
Reduces manual effort in writing repetitive test cases and speeds up the testing process.

### 4. Professional Test Case Format
Generates test cases with:
- Test Case ID
- Test Scenario
- Preconditions
- Test Steps
- Expected Results

### 5. Excel Export Support
Allows users to export generated test cases into Excel format for easy sharing and documentation.

### 6. User-Friendly Interface
Simple and responsive frontend interface built using HTML, CSS, and JavaScript for smooth user interaction.

### 7. Backend API Integration
Backend developed using Flask/Python to handle AI processing, API calls, and test case generation logic.

### 8. Database Storage
Uses MongoDB to store requirements and generated test cases for future access.

### 9. Prompt Engineering
Custom AI prompts are designed to generate accurate, structured, and QA-focused outputs.

### 10. Faster Test Design
Helps reduce manual test case writing time and improves QA productivity.
